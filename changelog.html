Version 1.4 Changes

  • P and p now paste with formatting. (Pasting without formatting is possible by doing Ctrl + P/p)
  • Made copy with yy or Y in normal mode more efficient
  • Fixed bug with x on multiline paragraphs
  • Fixed bug with r on multiline paragraphs or at the end of a line
  • Fixed bug with a on multiline paragraphs
  • Fixed bug with A at the end of a file in normal mode
  • Fixed bug with e/E jumping at the end of a file in normal mode
  • Fixed bug with w/W not going to the end of the file when pressed repeatedly
  • Fixed bug with I in normal mode not always going to the start of the line
  • Fixed behavior when using cc, when on an empty line the line doesn't get deleted with cc
