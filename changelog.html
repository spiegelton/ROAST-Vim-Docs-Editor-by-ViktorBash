Changelog

Support, Feature Requests: support@vimfordocs.com

Version 1.8 Changes

  • When you click and drag with your mouse in normal mode, Vim automatically changes to visual mode.
  • Added search shortcuts with Ctrl + f on Windows and Cmd + f on Mac.
  • Added search shortcut with / (all devices)

Version 1.7 Changes

  • Changed pricing options (for the better!): Purchasing options are now yearly or lifetime license (instead of just having a monthly option)

Version 1.6 Changes

  • A lot of bug fixes for a lot of keybindings in normal and visual mode. In coming updates there's going to be minimal bug fixes now (since the large majority are gone), and focus will be put more on shipping out new keybindings.
  • Redo functionality with Ctrl + r
  • Page up and Down functionality with Ctrl + u and Ctrl + d
  • In visual mode you can now bold, italicize, and strike through text with the regular Google Docs shortcuts.

Version 1.5 Changes

  • Ctrl + c also can be used to return to normal mode

Version 1.4 Changes

  • In general, a lot of bugs were fixed for the cursor movement, entering insert mode, and editing keybindings. Next update will be focused on solving bugs in visual mode.
  • P and p now paste with formatting. (Pasting without formatting is possible by doing Ctrl + P/p)
