Version 1.17 Changes

  • Added in db and cb
  • Added in de and ce
  • Redid { and } in normal mode to work properly
  • Revamp pasting (various bugfixes)
  • Various bugfixes
  • Fixed bugs with Replace Mode
  • Fixed bugs with Visual Line Mode
  • Fixed bugs with J
  • Added Ctrl+Shift+F shortcut to toggle hiding the menus (Native Google Docs shortcut)
  • Fixed bugs with shortcut for displaying word count
  • Added s and S in visual mode
  • Added s and S in visual line mode
  • Added Enter in normal mode
  • Fixed bug with UI for what mode you're in being covered by sidebar

Version 1.16 Changes

  • Added zz, zt and zb commands
  • Pricing change (for better)

Version 1.15 Changes (Major Update)

  • Added f, t, F and T keybindings to normal mode (searching for characters)
  • Added replace mode (use R in normal mode to access)
  • Overhauled visual line mode to work better
  • Added J for joining lines together
  • Added num support for yanking in normal mode (yy and Y)
  • Added improvements/bug-fixes for switching to visual mode using the mouse in normal mode
  • Added improvements/bug-fixes for r command
  • Added Spacebar keybinding in normal and visual mode
  • Pricing change
  • Other various bugfixes

Version 1.14 Changes

  • Added indent/outdent keybindings for normal, visual and visual line modes
  • Implement uppercase/lowercase keybindings for visual and visual line modes
  • Hotfix for bug with 'r' (replace command)
  • Minor improvement for visual mode cursor logic

Version 1.13 Changes (Major Update)

  • Added remapping support for all keybindings
  • Added support for a lot more native Google Docs shortcuts to work in normal/visual/visual line modes (increase/decrease font, superscript, subscript, etc)
  • Added remapping support for native Google Docs shortcuts
  • Minor improvements to pasting keybindings
  • Added option for many keybindings to cut to clipboard (dd, x, cc, etc) in normal, visual and visual line modes
  • Speed up and fix bugs with "x" keybinding
  • Fix bug with mouse detection to go to normal mode
  • Generally refactored codebase, making it easier to add more features in the future

Version 1.12 Changes

  • Added s in normal mode (same as pressing xi)
  • Added S in normal mode (same shortcut as cc)
  • Fixed bug with { not working in Mac visual mode
  • Fixed bugs with diw , ciw , daw and caw on Windows when on the last character in a line
  • Fixed bugs with dw and cw deleting improperly when on a space
  • Fixed bugs with dd and cc on Mac
  • Fixed bug with $ in visual mode on Mac
  • Let Ctrl + 1 to 9 pass through as shortcut (Cmd +1 to 9 on Mac)
  • Let Google Docs native shortcuts for bullet points and text style (Normal text, heading 1, heading 2, etc) pass through in normal and visual mode
  • Added Backspace as shortcut

Version 1.11 Changes

  • Added support for cw
  • Added support for daw and caw
  • Revamped diw and ciw to be more accurate (specifically in handling whitespace)

Version 1.10 Changes

  • Added support for arrow keys for navigation
  • Added support for left, center and right align text in either normal or visual modes (and also justify text)
    • Ctrl or Command (Mac) + Shift + L - Left align text
    • Ctrl or Command (Mac) + Shift + E - Center align text
    • Ctrl or Command (Mac) + Shift + R - Right align text
    • Ctrl or Command (Mac) + Shift + J - Justify text
  • Added support for Ctrl + a for Windows and Command + a on Mac for selecting all the text
  • Fixed bugs with dw and made it more robust

Version 1.9 Changes

  • Added dw support
  • Made better system for changing Vim modes based on mouse movements
  • Fixed bug with o being used near the end of a file on windows
  • Fixed small bug with A when used on second-to-last line in a file
  • Fixed bug with r not working with some special keys like "$", "%", "*", etc
  • Fixed bug with dd on Mac
  • Added number support for O and o
  • Added number support for x

Version 1.8 Changes

  • Added { and } for traversing paragraphs at a time (visual and normal mode)
  • When you click and drag with your mouse in normal mode, Vim automatically changes to visual mode.
  • When you click once somewhere with your mouse in visual mode, Vim automatically changes to normal mode.
  • Added search shortcuts with Ctrl + f on Windows and Cmd + f on Mac.
  • Added search shortcut with / (all devices)

Version 1.7 Changes

  • Changed pricing options (for the better!): Purchasing options are now yearly or lifetime license (instead of just having a monthly option)

Version 1.6 Changes

  • A lot of bug fixes for a lot of keybindings in normal and visual mode. In coming updates there's going to be minimal bug fixes now (since the large majority are gone), and focus will be put more on shipping out new keybindings.
  • Redo functionality with Ctrl + r
  • Page up and Down functionality with Ctrl + u and Ctrl + d
  • In visual mode you can now bold, italicize, and strike through text with the regular Google Docs shortcuts.

Version 1.5 Changes

  • Ctrl + c also can be used to return to normal mode

Version 1.4 Changes

  • In general, a lot of bugs were fixed for the cursor movement, entering insert mode, and editing keybindings. Next update will be focused on solving bugs in visual mode.
  • P and p now paste with formatting. (Pasting without formatting is possible by doing Ctrl + P/p)
