Ctrl, Shift, Alt and Cmd key options don't matter for keybindings that are set to more than one character (for example: diw )

You can't set function keys (F1-F12) or modifier keys (Control, Shift, Alt, Command/Meta) as keybindings by themselves

Insert mode keybindings can be at most 1 key long

Native Google Docs Shortcuts refer to built-in Google Docs commands. Changing these keybindings affects normal, visual and visual line mode.

It is recommended that any keybindings to escape/exit (ex: Ctrl-c, Esc), you keep at most 1 character long
