Tip: Prefix a command with a number to repeat it. For example, 4j moves down 4 lines.

Cursor movement

  • h - move cursor left
  • j - move cursor down
  • k - move cursor up
  • l - move cursor right
  • { - move up a paragraph
  • } - move down a paragraph
  • w/W - jump forwards to the start of a word
  • e/E - jump forwards to the end of a word
  • b/B - jump backwards to the start of a word
  • 0 - jump to the start of the line
  • $ - jump to the end of the line
  • gg - go to the first line of the document
  • G - go to the last line of the document
  • Ctrl + d - Page Down
  • Ctrl + u - Page Up
  • Ctrl + f - Search (Windows)
  • Cmd + f - Search (Mac)
  • / - Search
  • Arrow Keys - move cursor around

Entering Insert Mode

  • i - insert before the cursor
  • I - insert at the beginning of the line
  • a - insert (append) after the cursor
  • A - insert (append) at the end of the line
  • o - open a new line below the current line
  • O - open a new line above the current line
  • Esc or Ctrl + c- exit insert mode

Editing

  • x - delete a single character
  • r - replace a single character
  • dd - delete entire line
  • cc - delete entire line and enter insert mode
  • d$ or D - delete to end of line
  • c$ or C - delete to end of line and enter insert mode
  • d0 - delete to start of line
  • c0 - delete to start of line and enter insert mode
  • dw - delete to end of word
  • cw - delete to end of word and enter insert mode
  • diw - delete entire word
  • ciw - delete entire word and enter insert mode
  • daw - delete entire word, including whitespace
  • caw - delete entire word and enter insert mode, including whitespace
  • yy or Y- yank (copy) a line
  • y$ - yank (copy) to end of line
  • y0 - yank (copy) to start of line
  • p - paste clipboard after cursor
  • Ctrl + p - paste without formatting clipboard after cursor
  • P - paste clipboard before cursor
  • Ctrl + P - paste without formatting clipboard before cursor
  • u or U - undo
  • Ctrl + r - redo

Visual Mode

  • v - start visual mode
  • V - start linewise visual mode (experimental support)
  • x - cut to clipboard the highlighted text
  • d - delete the highlighted text
  • D - delete all lines of the highlighted text
  • c - delete the highlighted text and enter insert mode
  • C - delete all lines of the highlighted text and enter insert mode
  • y - yank (copy) the highlighted text
  • p - paste over the highlighted text after highlight
  • P - paste over the highlighted text before highlight
  • I - exit into insert mode at the beginning of the highlighted text
  • A - exit into insert mode at the end of the highlighted text
  • Ctrl + f - Search (Windows)
  • Cmd + f - Search (Mac)
  • / - Search
  • Arrow Keys - move cursor around
  • Esc or Ctrl + C - exit visual mode

    • Google Docs Native Visual Shortcuts: (Windows)

  • Ctrl + b - Bold Text
  • Ctrl + i - Italicize Text
  • Alt + Shift + 5 - Strike Through Text

    • Google Docs Native Visual Shortcuts: (Mac)

  • Command + b - Bold Text
  • Command + i - Italicize Text
  • Command + Shift + X - Strike Through Text
